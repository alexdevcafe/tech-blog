According to [MDN Web Docs](https://developer.mozilla.org/en-US/docs/Web/HTTP/Status), an HTTP response status code is an indication of whether a specific HTTP request has been successfully completed or not.

Additionally, these responses are classified into the following groups:

- **100-199** informational responses
- **200-299** successful responses
- **300-399** redirection messages
- **400-499** client error responses
- **500-599** server error responses

NodeJS has a handy native HTTP module that allows transferring data over the Hypertext Transfer Protocol (HTTP).

Assuming that you already have [NodeJS](https://nodejs.org/en/) runtime installed on your machine, use the command below to view the HTTP status codes in your terminal.

```js
node --print http.STATUS_CODES
```
